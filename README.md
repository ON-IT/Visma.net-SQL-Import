# Visma.net SQL Import

Convert from **any** ERP system that uses **SQL Server** using the Visma.net SQL Import from ON IT AS.

## Currently supported

 * Customers
 * Suppliers
 * Inventory

## ERP Systems

 * [Visma Global](Visma Global)
 * [Visma Business](Visma Business)
 * [Uni Micro](Uni Micro)
 * [Flat File](Flat File)
